Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors web game built using HTML, CSS, and JavaScript. The game allows users to play against the computer, which randomly selects its move. The application keeps track of wins, losses, and ties using localStorage, so the score persists even after refreshing the page.

Features

  -Play Rock, Paper, Scissors against the computer
  -Random computer move generation
  -Score tracking (Wins, Losses, Ties)
  -Score saved using localStorage
  -Reset score functionality
  -Interactive UI with icons for each move

Technologies Used

  -HTML5 – structure of the application
  -CSS3 – styling, layout, and animations
  -JavaScript – game logic and score tracking

How to Use

Prerequisites:
  -A modern web browser (Chrome, Edge, Firefox, Safari)

Steps:
  Clone the repository:
  git clone https://github.com/Blessing-D3velop/Rock-Paper-Scissors.git

Open the index.html file in your browser.

Click Rock, Paper, or Scissors to play against the computer.
